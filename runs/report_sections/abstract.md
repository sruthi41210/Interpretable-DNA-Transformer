# Abstract

The identification and interpretation of regulatory elements within the non-coding human genome remain a fundamental challenge in computational biology. While large-scale deep learning foundation models have achieved state-of-the-art results in sequence classification, these often function as complex "black boxes," lacking the transparency required to provide actionable causal biological insights. This research presents the **TinyDNAEncoder**, an efficient and lightweight Transformer-based architecture specifically optimized for the classification of human promoter and enhancer sequences. By integrating **Attention with Linear Biases (ALiBi)**, the model demonstrates remarkable data efficiency and length extrapolation capabilities, maintaining high performance (AUROC 0.89, PR-AUC 0.92) even when trained on significantly reduced data fractions compared to standard architectures.

To address the "interpretability gap," a robust multi-method framework is implemented that contrasts global sequence statistics with local causal impacts. This pipeline utilizes k-mer enrichment alongside systematic **In-Silico Mutagenesis (ISM)** to effectively distinguish between correlative background features and true causal regulatory motifs. The analysis successfully identified the **TFAP2A** binding motif as a primary causal driver of prediction scores, exhibiting a **1.51x enrichment** in high-importance sequence windows with robust statistical significance (**p = 1.04e-05**). Conversely, traditional k-mer analysis primarily highlighted background architectural elements like **ZNF610**, which are prevalent in GC-rich promoters but demonstrate minimal causal impact on model outputs. By providing a biologically grounded approach to deep learning, this work offers a scalable framework for discovering functional elements and interpreting non-coding disease variants, bridging the critical divide between high-performance sequence modeling and functional genomics.

**Keywords:** DNA transformers, interpretability, regulatory motifs, in-silico mutagenesis, ALiBi, functional genomics, TFAP2A.
